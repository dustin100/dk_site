Under the hood

This site is a lightweight HTML build with structured SCSS and BEM-style components. No heavy frameworks here — just thoughtful markup, accessible styles, and a dash of vanilla JS for polish.

  • HTML + semantic structure
  • SCSS with design tokens (spacing, type, layers, icons)
  • BEM naming for predictable components
  • ES modules for tiny interactivity (no runtime dependency)
